# go-gen-openapi-dep-detector Configuration

Dependency detector for OpenAPI code generation

> Full OpenAPI specification: [spec.openapi.yaml](../spec.openapi.yaml)

## Fields

### `resolveRefs`

- **Type:** `boolean`
- **Required:** No
- **Description:** Whether to resolve external references

### `rootDir`

- **Type:** `string`
- **Required:** No
- **Description:** Root directory for resolving relative paths

### `specSources`

- **Type:** `array of string`
- **Required:** No
- **Description:** OpenAPI spec file paths to scan
