# go-lint Configuration

Run golangci-lint for Go code linting

## Fields

### `args`

- **Type:** `array of string`
- **Required:** No
- **Description:** Additional arguments to pass to golangci-lint (optional)

### `config`

- **Type:** `string`
- **Required:** No
- **Description:** Path to golangci-lint config file (optional)
